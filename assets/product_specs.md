# Product Specifications - E-Shop Checkout

## Discount Code Feature
- The discount code **SAVE15** applies a 15% discount to the total cart value
- Invalid discount codes should display an error message
- Discount should be applied before calculating the final total

## Shipping Options
- **Standard Shipping**: Free of charge
- **Express Shipping**: Costs $10 and must be added to the final total

## Product Catalog
1. **Laptop** - $999.99
2. **Wireless Mouse** - $29.99
3. **Keyboard** - $79.99

## Cart Functionality
- Users can add items to cart using "Add to Cart" buttons
- Users can modify quantity of items in the cart
- Cart total must update dynamically when quantities change
- Cart must display individual item totals and overall total

## Payment Methods
- Credit Card
- PayPal

Both payment methods should be selectable via radio buttons.
